HTML - OVERVIEW:

HTML stands for Hypertext Markup Language, and it is the most widely used language to write Web Pages.

  • Hypertext refers to the way in which Web pages (HTML documents) are linked together. Thus, the link available on a webpage is called Hypertext.
  • As its name suggests, HTML is a Markup Language which means you use HTML to simply "mark-up" a text document with tags that tell a Web browser how to structure it to display.
  • Current Version of Html we are using is HTML5. HTML is third generation language followed by SGML and GML.

HTML-STRUCTURE:

FORMATTING - TAGS:

These tags are used to format the words/text.

  1. Pre tag :

    The Pre tag is used to display preformatted text exactly as it appears in the HTML code. It preserves both spaces and line breaks, making it useful for displaying code snippets, ASCII art, or any text that needs to maintain its original formatting.

  2. b tag :

    The b tag is used to apply a bold formatting to the enclosed text. However, it doesn't necessarily carry any semantic meaning. It's used mainly for stylistic purposes when you want to make text visually bold.

  3. Strong tag :

    Similar to the b tag, the strong tag is used for applying bold formatting. However, it also conveys semantic importance. It's used to indicate that the enclosed text is of strong importance or emphasis, both visually and for screen readers.

  4. i tag :

    The i tag is used to apply italic formatting to the enclosed text. Like the b tag, it doesn't necessarily carry semantic meaning. It's used for stylistic purposes when you want to make text visually italicized.

  5. em tag:

    Similar to the i tag, the em tag is used for applying italic formatting. However, it also conveys semantic importance. It's used to indicate that the enclosed text is emphasized or important, both visually and for screen readers.

  6. s tag :

    The s tag is used to apply a strikethrough effect to the enclosed text. It's often used to visually indicate that the text has been deleted or is no longer relevant.

  7. del tag :

    The del tag is used to represent content that has been deleted from a document. It typically renders with a strikethrough effect, indicating that the content is no longer valid or relevant.

  8. ins tag :

    The ins tag is used to represent content that has been inserted into a document. It often renders with an underline or a highlighted background, indicating that the content is new or recently added.

  9. sup tag :

    The sup tag is used to format text as superscript, raising it above the baseline. It's commonly used for footnotes, mathematical exponents, and other instances where text needs to be positioned above the normal line.

  10. sub tag :

    The sub tag is used to format text as subscript, lowering it below the baseline. It's often used for chemical formulas, mathematical subscripts, and other instances where text needs to be positioned below the normal line.
